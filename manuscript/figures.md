## Figure 1 — Dissonance Map Heatmap

**Artefact:** `outputs/dissonance.csv` (tabular; rendered as heatmap)

**Additional plotting code required:** Yes. `scripts/run_dfs.py` writes the CSV
but does not currently render the 6×6 heatmap. A rendering script
`scripts/render_figures.py` is a planned TODO (see `manuscript/README.md`). The
heatmap should be a symmetric matrix of the 15 pairwise dissonance scores d_ij,
with TOPCAT-Russia/Georgia highlighted via a coloured border or annotation.
Suggest: `seaborn.heatmap` with `annot=True`, `fmt=".2f"`, and a diverging
colourmap (blue = low, red = high) with midpoint at d = 0.5.

**Caption draft:** Pairwise dissonance scores for all 15 trial pairs in the
MRA-HFpEF evidence base. Each cell reports the z-score for the hypothesis of
equal log-hazard ratios between the corresponding pair. The TOPCAT-Americas versus
TOPCAT-Russia/Georgia comparison (d = 1.56) is the dominant dissonance. All
other pairs involving TOPCAT-Russia/Georgia produce scores above 1.30 (yellow–red),
reflecting that arm's systematic divergence from the remaining five boundary
conditions. All non-Russia/Georgia pairs cluster below d = 0.55 (blue).

## Figure 2 — Effect Field over LVEF × eGFR

**Artefact:** `outputs/field_lvef_egfr.png`

**Additional plotting code required:** No. `scripts/run_dfs.py` generates this
figure. It shows GP posterior mean and posterior standard deviation as side-by-side
heatmaps over the LVEF × eGFR plane at fixed anchors (age 72 y, K⁺ 4.3 mmol/L,
DM fraction 0.35, MR-occupancy 1.0, adherence proxy 0.85). Trial anchor points
are overlaid as labelled markers.

**Caption draft:** Gaussian process posterior effect field for the primary
composite endpoint over left ventricular ejection fraction (LVEF) and estimated
glomerular filtration rate (eGFR), at fixed population anchors (age 72 years,
baseline potassium 4.3 mmol/L, diabetes prevalence 35%, MR-occupancy equivalent
to spironolactone 25 mg/day, adherence proxy 0.85). Left panel: posterior mean
log-hazard ratio (blue = benefit, red = harm). Right panel: posterior standard
deviation. Trial anchor points are overlaid as labelled crosses. The effect field
is more negative (greater benefit) at higher adherence proxy values (not shown
here; see Figure 5 for the adherence slice). Note the near-uniform benefit signal
in the left panel across the LVEF × eGFR range once adherence proxy is held at
0.85, contrasting with the near-null or harmful signal that emerges when the
adherence proxy is set to 0.40 (TOPCAT-Russia/Georgia level).

## Figure 3 — Mind-Change Price Map

**Artefact:** `outputs/mind_change_price.csv` (tabular; requires rendering)

**Additional plotting code required:** Yes. `scripts/run_dfs.py` writes the CSV
with columns (lvef, egfr, mcp_primary_composite, ...) but does not render the
heatmap. The rendering script should produce a heatmap in the same LVEF × eGFR
space as Figure 2. Values below 100 patient-years should be annotated as "fragile
recommendation" regions; values above 1,000 patient-years as "settled".

**Caption draft:** Mind-change price map for the primary composite endpoint
prescribing threshold (log-HR < log(0.90)). Each cell reports the pseudo-patient-years
of hypothetical null-result evidence (log-HR = 0) that would shift the posterior
recommendation past the non-recommendation threshold. Low values (dark regions)
indicate regions of the covariate space where the current recommendation is
fragile and sensitive to a single new trial; high values indicate regions where
the recommendation is robust. The high-eGFR, high-LVEF region (corresponding
approximately to the TOPCAT-Americas population) has a higher mind-change price
than the low-eGFR region, reflecting the stronger GP evidence density from
FIDELIO and FIGARO in that sub-space.

## Figure 4 — Feasibility Region

**Artefact:** `outputs/feasibility_mask.csv` (Boolean grid; requires rendering)

**Additional plotting code required:** Yes. The CSV contains Boolean cells
indicating whether the 95% predictive interval excludes the no-recommendation
threshold (log-HR = 0). The rendering script should overlay the feasibility
region as a transparent shading on the LVEF × eGFR grid, with the feasible region
(where recommendation is defensible without additional trials) shown in green and
the infeasible region in grey.

**Caption draft:** Feasibility region overlay for the primary composite endpoint.
A covariate location is classified as feasible (green) if the GP posterior 95%
predictive interval excludes log-HR = 0 in the direction of benefit. Grey regions
indicate insufficient current evidence to exclude the possibility of a null or
harmful treatment effect. The feasibility region covers the majority of the
clinical covariate space at adherence proxy 0.85, consistent with the conclusion
that MRA therapy is defensible without additional trials in adherent populations
across the tested LVEF and eGFR range.

## Figure 5 — ΔK⁺ Safety Field

**Artefact:** `outputs/safety_delta_k_lvef_egfr.png`

**Additional plotting code required:** No. `scripts/run_dfs.py` generates this
figure. It shows GP posterior mean and SD of ΔK⁺ over the LVEF × eGFR plane
with per-trial observed ΔK⁺ markers overlaid.

**Caption draft:** Posterior mean serum potassium change (ΔK⁺, mmol/L) as a
function of LVEF and eGFR from the safety Gaussian process field constrained by
the k_sign law (ΔK⁺ ≥ 0 wherever MR-occupancy > 0). All values are positive,
confirming that the pharmacological conservation law is non-binding across the
covariate space. Per-trial observed ΔK⁺ values are overlaid as labelled markers:
TOPCAT-Russia/Georgia has the lowest observed value (+0.02 mmol/L), consistent
with near-zero receptor occupancy from non-adherence; FIDELIO-DKD HF-subgroup
has the highest (+0.23 mmol/L), consistent with potentiated class effect at lower
baseline eGFR (median 44 mL/min/1.73m²). The k_sign constraint was non-binding
at all 50 virtual observation points (minimum posterior ΔK⁺ = +0.047 mmol/L).

## Figure 6 (Optional) — Forest Plot Comparison: DFS LOO vs Naive MA Pool

**Artefact:** None currently. Requires new plotting code.

**Additional plotting code required:** Yes, entirely new. This figure would show:
- A conventional forest plot of the six trials' log-HR estimates with 95% CI.
- An additional row showing the naïve REML random-effects pooled estimate with
  95% CI and prediction interval.
- A final row showing the DFS LOO prediction for FINEARTS-HF at its anchor
  covariate, to illustrate that DFS produces a tighter interval than the REML PI
  while still containing the observed value.

**Caption draft:** Forest plot comparing the DFS leave-one-out prediction for
FINEARTS-HF with a conventional random-effects meta-analytic prediction interval.
The six trial-level estimates are shown above (squares with 95% CI; size
proportional to inverse variance). The bottom rows show the REML pooled estimate
and 95% prediction interval (using TOPCAT-full rather than the split used in
DFS, for fair comparison) and the DFS LOO 95% predictive interval. The DFS
predictive interval (width 0.25 log-HR units) is substantially narrower than
the REML prediction interval, reflecting the gain from modelling adherence
as an explanatory covariate rather than absorbing it into τ².
